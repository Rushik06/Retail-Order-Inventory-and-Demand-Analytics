#  Retail Order Inventory & Demand Analytics

A full-stack monorepo application for retail order management, inventory tracking, authentication, and demand analytics.

Built using modern web technologies with Docker-based containerized architecture.

---

## Tech Stack

**Frontend**
- React
- Vite
- TailwindCSS
- Axios
- Zustand

**Backend**
- Node.js
- Express
- Sequelize ORM
- JWT Authentication

**Database**
- PostgreSQL

**DevOps**
- Docker
- Docker Compose
- Nginx (Reverse Proxy)
- pnpm Workspace (Monorepo)
